Xi Project - Milošův Bar


Funkce webové stránky:

Milošův Bar je webová aplikace pro imaginární bar, která poskytuje uživatelům informace o barmanech, nabízí menu s drinky, umožňuje rezervace stolů a poskytuje kontaktní údaje. Uživatelé se mohou také zaregistrovat a přihlásit se k přístupu do zabezpečených částí aplikace.


Použité technologie:

HTML, CSS a JavaScript pro frontendovou část
Bootstrap CSS framework pro responzivní design a stylování stránek
jQuery knihovna pro animace, eventy a generaci kartiček jednotlivých drinků
PHP pro backend a implementaci přihlášení, registrace a správy databáze
MySQL databáze pro ukládání dat
API pro získávání informací o menu drinků


Jak nainstalovat a spustit:

Naklonujte si tento repozitář z GitHubu na své lokální zařízení.
Nastavte si lokální webový server (např. Apache nebo Nginx) a zajistěte podporu PHP a MySQL.
Vytvořte prázdnou MySQL databázi a importujte do ní přiložený SQL soubor database.sql, který vytvoří potřebné tabulky.
Otevřete soubor config.php a upravte připojení k databázi pomocí příslušných údajů.
Spusťte aplikaci otevřením domovské stránky index.html ve webovém prohlížeči.
